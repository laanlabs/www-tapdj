# TapDJ - Static Site

This is a static version of the TapDJ iPhone DJ app website, converted from the original WordPress site.

## About TapDJ

TapDJ is the Ultimate DJ App for iPhone and iPod Touch. Mix and Scratch your iPod Library, add effects and equalize your tracks. Record your mixes and discover new music.

### Features

- **iPod Access** - One click integrated library
- **Real Scratching** - Responsive low-latency scratching
- **Effects + Equalizer** - Make your mix sound great
- **Cue Points + Loops** - 3 Loopable Cue points per deck
- **Mix Recording** - Save and transfer over USB
- **Sampler + Sounds** - Pre-loaded with hits and sound FX
- **GyroScratch!** - Use the gyroscope to scratch
- **Music Discovery** - Share and rate top tracks + mixes
